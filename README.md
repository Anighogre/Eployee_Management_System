# Eployee_Management_System
This employee management system is made by using java 
<br>
In this employee management system firstly make a page where one button is shown.
<br>
When you click on this button you have to see a login page where the admin can login using their username and password.
<br>
After the successfull login the homepage will be seen on this home page you have to see four options or buttons where you can add employee, view employee, update employee and delete employee
